## ğŸ”„ Component Interactions

### InstagramGraphAPI (Core)
**Responsibilities:**
- Communicate with Meta Graph API
- Handle two-step posting process
- Manage access tokens
- Fetch comments and insights

**Interactions:**
- Used by: All other components
- Calls: Meta Graph API v21.0

### Caption Generator
**Responsibilities:**
- Generate human-like captions
- Apply Brandon's voice
- Create natural variance
- Occasional typos

**Interactions:**
- Uses: OpenAI GPT-4 API
- Called by: Main automation script
- Input: Product context
- Output: Natural caption text

### Posting Scheduler
**Responsibilities:**
- Manage posting schedule
- Add random delays
- Track posting history
- Detect patterns

**Interactions:**
- Uses: node-cron
- Calls: Main automation script
- Stores: Posting history JSON

### Product Queue
**Responsibilities:**
- Sync with Printful
- Prioritize products
- Track promotion history
- Rotate products

**Interactions:**
- Uses: Printful API
- Called by: Main automation script
- Stores: Product queue JSON

### Engagement Bot
**Responsibilities:**
- Monitor comments
- Generate replies
- Filter spam
- Track engagement

**Interactions:**
- Uses: InstagramGraphAPI + GPT-4
- Called by: Webhook handler + Scheduler
- Stores: Engagement tracking JSON

### Analytics Tracker
**Responsibilities:**
- Track post performance
- Generate UTM parameters
- Calculate metrics
- Export reports

**Interactions:**
- Uses: InstagramGraphAPI
- Called by: Main automation script
- Stores: Analytics JSON
- Exports: CSV reports
