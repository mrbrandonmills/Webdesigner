### üü° MEDIUM #2: Book Unlock API Route Missing
**File:** Referenced in `/Volumes/Super Mastery/Webdesigner/components/book-reader.tsx:52` and `/components/pdf-book-viewer.tsx:52`
**Severity:** MEDIUM
**Impact:** Users cannot purchase/unlock books

**Problem:**
Both book reader components attempt to call `/api/books/unlock` (lines 52-62 in both files) but this API route doesn't exist.

**Evidence:**
```bash
$ find app/api -name "*book*"
# No results - no books API routes exist
```

**Steps to Reproduce:**
1. Navigate to `/writing/books/block-b` or `/writing/books/block-c`
2. Click "Unlock for $5" button
3. API call to `/api/books/unlock` will fail with 404
4. Browser console will show error
5. User cannot unlock content

**Fix Required:**
Create `/app/api/books/unlock/route.ts` with Stripe checkout integration:

```typescript
import { NextRequest, NextResponse } from 'next/server'
import Stripe from 'stripe'

const stripe = new Stripe(process.env.STRIPE_SECRET_KEY!)

export async function POST(request: NextRequest) {
  const { bookId, title, price } = await request.json()

  // Create Stripe checkout session
  const session = await stripe.checkout.sessions.create({
    payment_method_types: ['card'],
    line_items: [{
      price_data: {
        currency: 'usd',
        product_data: { name: title },
        unit_amount: price * 100, // Convert to cents
      },
      quantity: 1,
    }],
    mode: 'payment',
    success_url: `${process.env.NEXT_PUBLIC_BASE_URL}/writing/books/${bookId}?unlocked=true`,
    cancel_url: `${process.env.NEXT_PUBLIC_BASE_URL}/writing/books/${bookId}`,
  })

  return NextResponse.json({ checkoutUrl: session.url })
}
```

**Related Issue:**
Need success callback to mark book as unlocked in localStorage or database.
