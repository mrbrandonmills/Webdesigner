## Recording Guidelines

### Technical Requirements
1. **Audio Format:** MP3, 192kbps minimum
2. **Sample Rate:** 44.1kHz
3. **Volume:** Normalized to -3dB
4. **Background:** Optional subtle ambient soundscape (see individual script notes)
5. **Fade In/Out:** 2-3 seconds at beginning and end

### Voice Direction
Each script includes specific voice direction notes:
- Pacing instructions
- Pause durations (marked in seconds)
- Tone and energy level
- Background music suggestions
- Special considerations

### Quality Control Checklist
- [ ] All pauses accurately timed
- [ ] Voice quality matches character description
- [ ] Background music (if used) doesn't overpower voice
- [ ] No audio clipping or distortion
- [ ] Consistent volume throughout
- [ ] Professional intro/outro if desired
- [ ] File properly named and tagged
