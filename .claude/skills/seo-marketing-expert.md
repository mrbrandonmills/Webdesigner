## SEO STRATEGY FRAMEWORK

### 1. KEYWORD RESEARCH & TARGETING

**High-Intent Buyer Keywords** (prioritize for affiliate products):
- Product-specific: "best [product name] review", "[brand] vs [competitor]"
- Problem-solution: "how to [solve problem]", "[problem] solution"
- Buying intent: "buy [product]", "[product] discount", "[product] price"
- Comparison: "[product A] vs [product B]", "alternatives to [product]"

**Meditation Store Keywords**:
- "guided meditation for [specific need]"
- "[benefit] meditation"
- "meditation for anxiety/sleep/stress relief"
- "best meditation app alternative"
- "meditation audio download"

**Implementation**:
```markdown
For each product/meditation, create:
1. Primary keyword (high volume, high intent)
2. 3-5 secondary keywords (related, medium volume)
3. Long-tail variations (specific, lower competition)
```

### 2. ON-PAGE SEO OPTIMIZATION

**Title Tags** (55-60 characters):
```
[Product Name] Review: [Key Benefit] | Brandon Mills
[Meditation Name] - Guided Audio for [Benefit]
```

**Meta Descriptions** (150-160 characters):
```
Discover why [product] is the #1 choice for [benefit]. Expert review, features, pricing & where to buy. [CTA]
```

**URL Structure**:
```
/shop/[product-slug]
/meditations/[meditation-slug]
/writing/essays/[essay-slug]
/writing/books/[book-slug]
```

**H1-H6 Hierarchy**:
```
H1: Primary keyword (1 per page)
H2: Section headers with secondary keywords
H3: Subsections
- Include keywords naturally
- Focus on user intent
- Answer questions directly
```

**Image Optimization**:
```
- Alt text: Descriptive with keywords
- File names: product-name-feature.jpg
- Compress for speed (<200KB)
- Use WebP format
- Lazy loading
```

### 4. TECHNICAL SEO CHECKLIST

**Site Speed** (Target: <2s load time):
- [ ] Enable Next.js image optimization
- [ ] Minimize JavaScript bundles
- [ ] Use CDN for static assets
- [ ] Enable gzip/brotli compression
- [ ] Lazy load images below fold

**Core Web Vitals**:
- [ ] LCP (Largest Contentful Paint) < 2.5s
- [ ] FID (First Input Delay) < 100ms
- [ ] CLS (Cumulative Layout Shift) < 0.1

**Mobile Optimization**:
- [ ] Responsive design
- [ ] Touch-friendly buttons (min 44x44px)
- [ ] Readable text without zooming
- [ ] No horizontal scrolling

**Schema Markup** (Add JSON-LD):
```javascript
// Product Schema
{
  "@context": "https://schema.org/",
  "@type": "Product",
  "name": "Product Name",
  "description": "Product description",
  "brand": {"@type": "Brand", "name": "Brand Name"},
  "offers": {
    "@type": "Offer",
    "price": "499.99",
    "priceCurrency": "USD"
  },
  "aggregateRating": {
    "@type": "AggregateRating",
    "ratingValue": "4.5",
    "reviewCount": "1200"
  }
}

// Article Schema
{
  "@context": "https://schema.org",
  "@type": "Article",
  "headline": "Article Title",
  "author": {
    "@type": "Person",
    "name": "Brandon Mills"
  },
  "datePublished": "2025-11-17"
}
```

**Sitemap & Robots.txt**:
```xml
<!-- sitemap.xml -->
<urlset>
  <url>
    <loc>https://brandonmills.com/shop</loc>
    <priority>1.0</priority>
    <changefreq>weekly</changefreq>
  </url>
</urlset>
```

## IMPLEMENTATION CHECKLIST

When working on SEO/Marketing tasks, always:

1. **Start with User Intent**
   - What problem are they trying to solve?
   - What do they need to know before buying?
   - What objections do they have?

2. **Optimize for Featured Snippets**
   - Use question headers
   - Provide concise answers (40-60 words)
   - Use lists and tables
   - Structure for voice search

3. **Mobile-First Always**
   - Test on actual devices
   - Ensure tap targets are large enough
   - Check readability
   - Verify CTA visibility

4. **Speed is Revenue**
   - Every 100ms delay = 1% lost sales
   - Optimize images ruthlessly
   - Minimize redirects
   - Use browser caching

5. **Track Everything**
   - UTM parameters for all campaigns
   - Event tracking for key actions
   - Heatmaps to understand behavior
   - A/B test continuously
