## Common Questions

### "Can I disable backgrounds on a specific page?"

Yes! In that page's layout or component:

```tsx
// Option 1: Don't import PageBackground (it's in root layout)
// Just let the root layout handle it

// Option 2: Override with transparent background
<div className="relative z-10 bg-black">
  {/* Your content - backgrounds won't show through */}
</div>
```

### "Can I use a different variant than the auto-selection?"

Yes! Import GeometricBackground directly:

```tsx
import GeometricBackground from '@/components/ui/geometric-background'

<GeometricBackground
  variant="smoke"
  intensity="medium"
  color="white"
/>
```

### "Do backgrounds affect page load speed?"

No! They're:
- Client-side components (no SSR overhead)
- Lazy-loaded with React
- Use SVG (no image downloads)
- Cached after first render

### "Are backgrounds accessible?"

Yes!
- `pointer-events-none` â†’ Don't block interactions
- Subtle opacity â†’ Don't affect text contrast
- Slow animations â†’ No seizure triggers
- Decorative only â†’ No ARIA needed
